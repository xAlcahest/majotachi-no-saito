Come applicare la patch italiana

  1. Apri Steam e vai alla tua libreria
  2. Fai clic destro su "Placeholder" nella barra laterale, quindi scegli "Gestisci" e infine "Sfoglia i file locali"
  3. Estrai il contenuto del file .zip della patch nella cartella che si è appena aperta, scegliendo di unire le cartelle e di sovrascrivere i file esistenti.
  4. Avvia il gioco e seleziona la lingua italiana nelle impostazioni.

Nota: La patch include SOLO i file contenenti i dati della lingua italiana. Per poterla utilizzare è necessario possedere la versione completa del gioco.
